# Raw Books


### Description: 

This is staging area for books that are being prepped for import.

Texts should be broken into paragraphs with the following format:

* Blank line before each paragraph
* Markdown style triple hash before subheaders like so:

 ### Approaching the Inevitable

* Markdown style double hash before section headers like so:

 ## Chapter 1: Release the Kraken

* Optional embedded page markers Ocean-style: 

   <p 123>

* Optional paragraph markers on their own line before paragraph with a hash and a colon:

 #3.3:




### Workflow

 After prepping a book using a text editor, use the ocean tools import script to convert the document into the next folder (2. formatted-notproofed) and then proceed with cleanup and proof-reading.
